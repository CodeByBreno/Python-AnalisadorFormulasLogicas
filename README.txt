# 🧠💡 Interpretador de Fórmulas Lógicas
Esse projeto foi desenvolvido para disciplina de Lógica para Computação, ministrada pelo professor Rosalvo Neto, no segundo semestre de 2023. Ele consiste em uma
aplicação gráfica que permite a inserção de fórmulas lógicas, que são analisadas para determinar se sua estrutura está correta (fórmula bem formulada) e posteriormente
pode-se gerar a tabela verdade equivalente da expressão.
